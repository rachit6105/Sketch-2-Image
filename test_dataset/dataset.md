# Folder Description
What kind of files are in each files
- Kaggle
> Contains the full **CUFS (CUHK Face Sketch)** dataset collection used for face photo–sketch research.

- dataset
> Dataset used in the **Face-Sketch-SCG** project. Downloadable via the provided `wget` command as a compressed `.tgz` archive.
> CUFS  
> AR  
> CUFSF  
> CUHK_AR  
> FID_Ref  
> vggface  
> WildSketch  
> XM2VTS  

- IIIT-D Face Sketch
> The **IIIT-Delhi Face Sketch Dataset**, containing paired face photographs and sketches used for sketch recognition and synthesis research you need license to use this dataset
